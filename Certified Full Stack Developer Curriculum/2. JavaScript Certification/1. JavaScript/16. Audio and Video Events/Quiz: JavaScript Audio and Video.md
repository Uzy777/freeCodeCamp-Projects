## Questions

1) What is the purpose of the Audio() constructor in JavaScript?

- Create a new audio element.
- Create a new audio file.
- Modify audio files.
- Play an existing audio file.

**Answer: Create a new audio element.**

---

2) Which method is used to begin playback of an audio or video element?

- play()
- start()
- resume()
- begin()

**Answer: play()**

---

3) What is the difference between pause() and play() methods for media elements?

- pause() automatically resets to the beginning while play() starts from where the audio was paused.
- pause() returns a Promise while play() does not.
- pause() cannot be resumed while play() can start from any position.
- pause() stops playback but maintains the current position while play() starts playback.

**Answer: pause() stops playback but maintains the current position while play() starts playback.**

---

4) Which audio format has the widest browser support?

- WAV
- MP3
- FLAC
- OGG

**Answer: MP3**

---

5) What is the purpose of the source element within an audio or video element?

- Specify multiple media sources for different formats.
- Link to an external audio processor.
- Specify the volume of the media.
- Add subtitles to the media.

**Answer: Specify multiple media sources for different formats.**

---

6) What are the two possible MIME types an MP4 format can have?

- audio/mp4 and video/mp4
- mp4/audio and mp4/video
- mp4/mpeg and mp4/stream
- audio/mpeg4 and video/mpeg4

**Answer: audio/mp4 and video/mp4**

---

7) What does a codec do in the context of audio and video?

- Increases the volume of the media.
- Converts media between analogue and digital formats.
- Changes the file format of the media.
- Encrypts the media for secure transmission.

**Answer: Converts media between analogue and digital formats.**

8) Which API provides a way to create, process, and analyze audio in web applications?

- Web Audio API.
- Audio Processing API.
- Sound Manipulation API.
- Media Stream API.

**Answer: Web Audio API.**

9) What is the purpose of the getUserMedia() method?

- Access the user's camera and/or microphone.
- Retrieve user preferences for media playback.
- Authenticate users with a media service.
- Download media files to the user's device.

**Answer: Access the user's camera and/or microphone.**

10) Which property of the HTMLMediaElement interface represents the current playback time?

- playbackPosition
- currentTime
- currentPosition
- time

**Answer: currentTime**

11) What does the following code snippet do?

```js
const audio = new Audio('song.mp3');
audio.loop = true;
```


- Plays the audio file once.
- Pauses the audio after one loop.
- Sets the audio to repeat indefinitely.
- Increases the volume of the audio.

**Answer: Sets the audio to repeat indefinitely.**

13) What does the canPlayType() method help you do?
