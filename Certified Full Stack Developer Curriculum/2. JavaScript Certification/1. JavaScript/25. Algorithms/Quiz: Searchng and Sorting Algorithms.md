Questions

Q1) Which search algorithm iterates through a list of items, checking each item from the beginning until the target item is found?

- Binary search
- Bubble search
- Merge search
- Linear search

**Answer: Linear search**

---

Q2) Which technique is used to break down a problem into smaller sub-problems?

- Break-down-and-solve
- Divide-and-solve
- Divide-and-conquer
- Break-down-and-conquer

**Answer: Divide-and-conquer**

---

Q3) What is the space complexity of the linear search algorithm?

- O(n)
- O(1)
- O(log n)
- O(n²)

**Answer: O(1)**

---

Q4) Which is more suitable for a larger list of items between binary search and linear search?

- Linear search
- None
- Both
- Binary search

**Answer: Binary search**

---

Q5) What is the space complexity of binary search?

- O(log n)
- O(n)
- O(1)
- O(n log n)

**Answer: O(1)**

---

Q6) What does the binary search algorithm return when it finds the target item?

- A boolean value of the target.
- The index of the target item.
- The value of the target item.
- The size of the list searched.

**Answer: The index of the target item.**

---

Q7) What does the linear search algorithm return if it doesn't find the target item?

- 0
- null
- -1
- undefined

**Answer: -1**

---

Q8) How does the merge sort algorithm work?

- It recursively splits the list into smaller items, then merges them in sorted order.
- It searches for an element by checking each item one by one.
- It swaps adjacent elements until the list is sorted.
- It builds a sorted list by repeatedly finding the smallest element.

**Answer: It recursively splits the list into smaller items, then merges them in sorted order.**

---

Q9) What is the condition for a binary search to work?

- The list must have an even number of elements.
- The list must contain only unique values.
- The list must contain no negative numbers.
- The list must be sorted.

**Answer: The list must be sorted.**

---

Q10) Which computer science approach is used to implement the merge sort algorithm?

- Divide-and-conquer
- Dynamic programming
- Greedy algorithm
- Tracing

**Answer: Divide-and-conquer**
