Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used for web development, data analysis, artificial intelligence, scientific computing, and more.

Python's syntax allows programmers to express concepts in fewer lines of code than possible in languages such as C++ or Java.

Here is an example of a simple Python program:


print("Hello, World!")

    • Python Basics

    Before diving into advanced topics, it's important to grasp the basics of Python. These include understanding variables, data types, and basic operations.

    Variables in Python are created when you assign a value to them. For example:

    
x = 5
y = "Hello"

    Python supports several data types such as integers, floats, strings, and booleans. Here's how you can use them:

    
a = 10       # integer
b = 3.14     # float
c = "Hello"  # string
d = True     # boolean

    • Data Structures

    Python provides several built-in data structures such as lists, tuples, sets, and dictionaries. These structures help you organize and manipulate data efficiently.

    A list in Python is an ordered collection of elements that can be of different types. Here's an example: 

    
my_list = [1, 2, 3, "apple", "banana"]
print(my_list[0])  # Output: 1

    Tuples are similar to lists but are immutable. You cannot modify them once they are created. For example: 

    
my_tuple = (1, 2, 3)
print(my_tuple[0])  # Output: 1

    • Functions

    Functions are blocks of reusable code that perform a specific task. They help in organizing code and avoiding repetition.

    Here's an example of a simple function in Python:

    
def greet(name):
    return f"Hello, {name}!"
    
print(greet("Alice"))  # Output: Hello, Alice!

    • Modules and Packages

    Modules and packages in Python allow you to organize your code into separate files and directories, making it easier to manage and reuse.

    A module is simply a Python file with a .py extension that contains functions, classes, or variables. You can import a module using the import statement:

    
import math
print(math.sqrt(16))  # Output: 4.0

    A package is a collection of modules in a directory that includes a special __init__.py file. Here's how you can create and use packages:

    
# Directory structure:
# mypackage/
#     __init__.py
#     module1.py
#     module2.py

# module1.py
def foo():
    print("foo")

# Using the package
from mypackage import module1
module1.foo()  # Output: foo
